## 2. Variables

### 2.1 What is a Variable?

A variable is a name that is used to store data/value in memory, and that value can change during program execution.

One-line definition:
  A variable is a container that holds data which can be changed while a program runs.

Example: 
- Variable is used to store data
- It acts like a container

## 3. String 

### Defination
- A string is a sequence of characters (letters, numbers, symbols, or spaces) enclosed in quotes (single ' ' or double " "). It is used to store text in a program.
- name = "Vinay"
- greeting = 'Hello, World!'Strings are immutable in Python (cannot change individual characters).

- You can perform operations like concatenation, slicing, and formatting.

## 3. Python loops:
 
### Defination: 
  A loop in Python is used to execute a block of code repeatedly, as long as a condition remains true or until the elements of a collection are exhausted.
